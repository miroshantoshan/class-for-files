# File Working Class (🇺🇸EN)
## The File class is used to work with files

## Primary use: creating and editing files

### Note
#### This is still a rough application, be careful!
    
### Attributes

    file_name : str
        file name
    create 
        file creation

### Methods

    create_file()
        Creates a file
    write_to_file(text="Your text")
        Writes information (text) to a file
    add_to_file(text="Your added text")
        Adds text without deleting previous content
    read_file()
        Reads a file and returns its contents
    delete_file()
        Deletes a file
    rename_file("New name")
        Renames a file
    copy_file("Copy path")
        Copies a file with its contents to a specified location




# Класс для работы с файлами (🇷🇺RU)
## Класс File используется для работы с файлами

## Основное применение - создание и редактирование файлов

### Note
####  Это ещё сырое приложение, будьте осторожны!)
    
### Attributes

    file_name : str
        название файла
    create 
        создание файла

### Methods

    create_file()
        Создаёт файл
    write_to_file(text="Ваш текст")
        Вносит в файл информацию(текст)
    add_to_file(text="Ваш добавленный текст")
        Вносит в текст дополнения, не удаляя прошлые
    read_file()
        Прочитывает файл и возвращает содержимое
    delete_file()
        Удаляет файл
    rename_file("Новое название")
        Переименовывает файл
    copy_file("Путь копии")
        Копирует файл с содержимым в указанное место
